# Random Walker

A simple random walker program written in C with SDL2.

![Display random walker program](./doc/50-Random-Walker.png)

Shout out to [Daniel Hirsch](https://youtu.be/ErA4U9WqNCE?si=AuBcQSrJtlYU50O4)

# Installation

Install [SDL2](https://wiki.libsdl.org/SDL2/Installation) in advanced.

```bash
make
```

# Usage

```bash
$ ./prog <walker_number>
```
